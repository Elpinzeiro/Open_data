Our Use Cases

We mainly analyzed two different use cases. One uses static data and the other dynamic

Static

In this case we used a type of information that remains unchanged over time, such as the accessibility of a classroom by a disabled student

Dynamic

On the contrary, the availability of places in a university library is information subject to rapid changes and therefore needs to be updated frequently.

>

Buildings - Static Use Case

Thanks to the leaflet library and the openstreetmap maps, it was possible to show the accessibility of the classrooms by simply querying the dataverse server which will respond with a json file

Libraries - Dynamic use case

If, on the other hand, we want to show content that is subject to rapid changes over time, such as, for example, the availability of seats in a library. It will be necessary to query an endpoint external to the dataverse server that has the updated values.
